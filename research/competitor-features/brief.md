# Research Brief: Spanish Construction Budget Tools

## Question
What are the core features of leading Spanish construction budget/estimation tools, and what gaps exist that Smeta could exploit?

## Scope
- 8 specific tools: Presto, Costesite, ARQBIM, PlanSwift, Buildxact, STACK, Cronoshare, Habitissimo
- For each: Core budgeting features, Export formats, Client-facing features, Reporting, Integration, Mobile, Collaboration
- Focus on Spanish market differentiators for Smeta

## Depth Mode
Standard (3-5 sub-agents, 1 follow-up round, 15+ sources)

## Date
2026-07-13

## Angles
1. **Presto (Grupo Precon)** - Features, pricing, market position
2. **Costesite** - Online budget tool features
3. **ARQBIM** - BIM-based budget features
4. **PlanSwift** - Takeoff and budgeting features
5. **Buildxact** - Project management + budgeting
6. **STACK** - Construction estimation features
7. **Cronoshare** - Budget presentation features
8. **Habitissimo** - Budget calculator features
