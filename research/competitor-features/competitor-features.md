# Competitor Features Analysis: Spanish Construction Budget Tools

> Research date: 2026-07-13 | 8 tools investigated | 4 valid tools, 2 marketplaces, 2 non-existent

---

## Executive Summary

Of the 8 tools researched, **2 do not exist** (Costesite, ARQBIM), **2 are marketplaces** (Cronoshare, Habitissimo) with no budgeting features, and **4 are actual budget/estimation tools** (Presto, PlanSwift, Buildxact, STACK ecosystem). The Spanish construction budget market is dominated by **BC3 (FIEBDC-3)** as the mandatory interchange format, with a clear split between legacy desktop tools and emerging cloud platforms.

**Key Smeta differentiators to target:**
1. Cloud-native with mobile-first design (only Trowel/Planhopper offer this today)
2. BC3 native support as a given, not an add-on
3. Client-facing portal with digital quote acceptance
4. AI-powered takeoff and estimation (Buildxact/PlanSwift are pioneering this)
5. Verifactu electronic invoicing compliance
6. Transparent, affordable pricing vs. Presto's ~508 EUR/yr or Trowel's 269 EUR/mo

## Tool-by-Tool Analysis

### 1. Presto (GRAITEC/RIB Software)

**Status**: Market leader in Spain, legacy desktop tool

| Category | Features |
|---|---|
| **Core Budgeting** | Modular system: base budgeting + 10 optional modules (BIM Oriented budgets/measurements, project management, temporal/economic/financial planning, procurement, execution management, Revit integration, DWG reading, QMASS, profitability, report customization, API). Physical quantities (not percentages) for estimation. EVM (Earned Value Method) tracking. 60+ predefined WBS templates. |
| **Export Formats** | Bidirectional Excel import/export (Excel2Presto add-in). MS Project export. BC3 format support (implied by market dominance). Report export to Excel. |
| **Client-Facing** | Not prominently featured. No client portal. Professional-to-professional tool. |
| **Reporting** | Advanced report customization (requires basic programming knowledge). Multidimensional construction-specific information (operations, activities, calendar, approval states, building agents). |
| **Integration** | COST-IT: bidirectional Presto-Revit integration. MS Project and Primavera integration. Native DWG reading. API module available. |
| **Mobile** | None. Windows-only desktop software. |
| **Collaboration** | Multi-user via LAN and internet. Agent-based data control (each agent controls their own data). No real-time concurrent editing. |
| **Pricing** | From 174 EUR (perpetual) or ~508 EUR/year subscription |
| **Market** | Spain and Latin America. Most widely known construction budget tool in Spain. |

**Gaps Smeta could exploit:**
- Windows-only, no cloud/mobile
- Expensive for full feature set
- Outdated UI ("interfaz anticuada")
- No client-facing portal
- No AI features
- Report customization requires programming

### 3. ARQBIM

**Status**: Does not exist as a software product

All entities named "ARQBIM" found online are architecture/BIM consulting firms (Peru, Chile, Colombia, Mexico). None produce budget software. The arqbim.com domain does not resolve. No app store listings, press releases, or product announcements exist.

### 4. PlanSwift (ConstructConnect)

**Status**: Leading US desktop takeoff/estimating tool, no Spanish market presence

| Category | Features |
|---|---|
| **Core Budgeting** | Digital takeoff from PDF/DWG plans. 18+ trade-specific assembly libraries with drag-and-drop. Real-time cost calculation. Custom formulas for trade-specific logic. Takeoff Boost AI (May 2026): Auto Takeoff (area/linear/count in <1 min), Auto Count (detects symbols across 10 pages in 1-2 min). |
| **Export Formats** | Live Excel integration (real-time sync, not just CSV). Email export. Network drive file sharing. |
| **Client-Facing** | Not prominently featured. No client portal. |
| **Reporting** | Customizable built-in reports for customers, suppliers, subcontractors, coworkers. |
| **Integration** | Excel (required locally). Outlook for project sharing. Plugin marketplace with trade-specific Starter Packs. Part of ConstructConnect platform (On-Screen Takeoff, Quick Bid, Digital Production Control). |
| **Mobile** | None. Windows-only desktop (10/11 Pro/Enterprise 64-bit). |
| **Collaboration** | Limited to Email Job feature and shared network drives. No real-time multi-user collaboration. |
| **Pricing** | $2,000/year subscription (includes support, updates, 2 hours training) |
| **Market** | Primarily US market. No Spanish localization found. |

**Gaps Smeta could exploit:**
- No mobile or cloud version
- No Spanish language/localization
- No client-facing features
- Limited collaboration (network drives only)
- Desktop-bound, expensive
- No BC3 support

### 5. Buildxact

**Status**: Cloud-based all-in-one platform for residential builders (UK/AU/US/CA/NZ)

| Category | Features |
|---|---|
| **Core Budgeting** | Digital takeoffs from PDF plans (80% faster claim). Templates and live supplier pricing. Customizable quote letters with digital signatures. Purchase orders, work orders. Cost tracking by job/category/item. Change orders. Timesheets. |
| **Export Formats** | Professional customizable quote letters (logo, specs, PC/PS breakdowns). Schedule exports. Accounting sync (Xero/QuickBooks/MYOB). Specific PDF/CSV export not documented. |
| **Client-Facing** | **Client Portal**: view project documents, accept quotes online with digital signatures, see variations with price change highlights, centralized messaging hub. |
| **Reporting** | Real-time profitability by job/cost category/item. Estimate-to-actual comparison. Additional layers via accounting integrations. |
| **Integration** | Xero, MYOB, QuickBooks (2-way sync). Deputy workforce management. Supplier Connection marketplace (live pricing). |
| **Mobile** | **Buildxact Onsite** app (iOS + Android): site diary with progress photos, job management, lead management, crew invitations via SMS. Free download; Pro/Master plan for sync. |
| **Collaboration** | Unlimited users on all plans. Crew/subcontractor invitations via mobile. RFQ system for subcontractor bid management. Supplier Connection for live pricing collaboration. |
| **Pricing** | Foundation GBP 89/mo, Pro GBP 160/mo, Master GBP 262/mo (annual). Unlimited users on all plans. |
| **Market** | UK, Australia, US, Canada, New Zealand. No Spanish market presence or language support found. |

**Gaps Smeta could exploit:**
- No Spanish language or market presence
- No BC3 support
- Reporting relies on external accounting tools
- No construction-specific Spanish compliance (Verifactu)
- AI features are add-on, not core

### 6. STACK / Spanish Market Ecosystem

**Status**: STACK is a US takeoff tool; the Spanish market has its own ecosystem

STACK itself is a US-focused cloud takeoff/estimating tool. More relevant for Smeta is the **Spanish construction software ecosystem** identified in research:

| Tier | Tools | Price Range |
|---|---|---|
| **Legacy Desktop** | Presto (GRAITEC), Arquimedes (CYPE) | 508+ EUR/yr |
| **Mid-price Cloud** | MedicionPro (from 29 EUR/mo), BrickControl (99 EUR/mo), Billdin | 29-99 EUR/mo |
| **Full-platform Cloud** | Trowel (269 EUR/mo for 5 users), Planhopper | 100-269 EUR/mo |

**Key market facts:**
- **BC3 (FIEBDC-3)** is the mandatory interchange format for all Spanish construction budget tools
- **Verifactu** (electronic invoicing) is emerging as a key compliance differentiator (Trowel, Planhopper support it)
- **Trowel** (by Holded, Cantabria) has the broadest feature stack: budgets, cost control, purchasing, invoicing, warehouse, blueprints, quality control, time tracking, planning, mobile apps
- **Planhopper** offers community-shared price libraries, BC3-to-Excel conversion, and Gantt planning
- **MedicionPro** is the price leader at 29 EUR/mo with BC3, certifications, invoicing, Gantt

### 7. Cronoshare

**Status**: Consumer-to-professional marketplace, NOT a budgeting tool

Cronoshare connects homeowners to professionals (1M+ registered, 500K+ services/year). The "presupuesto" feature is a **request flow**: client submits form, up to 4 professionals respond with quotes. No budgeting tools, export formats, reporting, collaboration, mobile apps, or integrations exist. Operates in Spain, Italy, Brazil, Mexico, Chile.

**Not a meaningful competitor** for Smeta's construction budgeting features.

---

### 8. Habitissimo

**Status**: Lead-generation marketplace, NOT a budgeting tool

Habitissimo has processed 6.49M+ budget requests with 490K+ professionals. It offers:
- **Price guides**: Static editorial content with average prices, ranges, city breakdowns (not interactive calculators)
- **Professional ROI calculator**: Measures lead purchase ROI (not construction budgeting)
- **Mobile app** (Android only, 500K+ downloads, 3.4★ rating)
- Operates in Spain, Portugal, Chile, Mexico

**No interactive construction budget calculator** for end users. No export, integration, or collaboration features.

**Not a meaningful competitor** for Smeta's construction budgeting features.

## Differentiation Opportunities for Smeta

### Must-Have Features (Market Gaps)
1. **Cloud-native + Mobile-first** — Only Trowel offers both in Spain; Presto/Arquimedes are desktop-only
2. **BC3 native support** — Mandatory for Spanish market; non-negotiable
3. **Client-facing portal** — Buildxact has this; no Spanish tool does well
4. **Verifactu electronic invoicing** — Emerging compliance requirement
5. **AI-powered estimation** — PlanSwift/Buildxact pioneering this; no Spanish tool offers it

### Differentiating Features
1. **Transparent pricing** — Undercut Trowel (269 EUR/mo) and Presto (~508 EUR/yr)
2. **Real-time multi-user collaboration** — Most tools offer file-sharing, not true collaboration
3. **Integrated reporting** — Don't rely on external accounting tools like Buildxact
4. **WYSIWYG contract/quote editor** — MedicionPro offers this; most don't
5. **Community price library** — Planhopper model; valuable for smaller firms
6. **Latin America expansion** — Presto dominates but is aging; cloud-native Smeta could capture this

### Avoid
- Desktop-only architecture (Presto/PlanSwift trap)
- Requiring programming knowledge for reports (Presto)
- Over-reliance on integrations for basic features
- Pricing that excludes small contractors (<5 employees)
